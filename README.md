# LSTM-Activity-Recognition
LSTM with Google BlazePose for Activity Recognition
using 2D-Video or Web-Camera.  
  
### Classify activities
STAND  
WALK  
RUN  
SIT  
GREET (UP HANDS)  
FALLDOWN  
CROSS ARMS (STAND)  
CROSS ARMS (SIT)  
PUSH UP  
CLAP
